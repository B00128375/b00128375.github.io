Facebook is a handy tool for connecting with people as it is easy to search for friends by name, number or email address. Its most important features are the group page function and the messenger app as the app displays messages over other apps meaning it is the most convenient to use at any point in time

Instagram is useful for sharing images and videos meaning lecturers will be able to post video tutorials on their page, there is also the story function which allows for temporary updates

Snapchat is a messaging app which also allows the creation of group chats, it is very easy to share photos and videos and there is also a story function to allow users to make temporary posts.

whatsapp is a messaging app which allows users to easily create group chats, images, videos and most file types can be shared through the app

Tumblr is a blogging site which allows users to make posts in a text, video, image or audio format.
